Win + R키를 눌러 실행 창을 열고, cmd를 입력한 후 엔터를 쳐 명령 프롬포트를 연 후 cd (파일 경로) 입력 후 엔터를 치고 pip install -r requirements.txt를 입력하고 엔터를 눌러 필요 모듈을 다운로드 할 수 있습니다.
베타 버전이라 버그가 상당히 많음
